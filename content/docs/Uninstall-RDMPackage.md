# Uninstall-RDMPackage

## SYNOPSIS

## SYNTAX

```
Uninstall-RDMPackage [[-Edition] <String>] [-Quiet] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Uninstall the installed version of Remote Desktop Manager

PS C:\> Uninstall-RDMPackage
